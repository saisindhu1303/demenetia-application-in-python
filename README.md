FOERGE – Forgetting Is Not Regretting Anymore

Project Overview
FOERGE is a mobile application designed to support cognitive health and memory improvement. The application provides brain training games, memory support through images, activity tracking, and medical record management to help users and caregivers manage daily life more effectively.

---

# Objectives
- Provide cognitive stimulation through interactive brain games.
- Help users recall memories using familiar images.
- Track daily activities and routines.
- Store medical records securely.
- Provide quick access to emergency contacts.
